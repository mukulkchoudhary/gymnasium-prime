🏛️ Gymnasium Prime
A Production-Grade Backend Ecosystem for Fitness Management
📖 Overview

Gymnasium Prime is a full-stack gym management system built from scratch to production readiness. This project serves as a comprehensive portfolio piece demonstrating mastery of the Django backend ecosystem, payment integration, and modern deployment practices.

The system handles user authentication, membership management, booking schedules, payment processing, and role-based dashboards for members, trainers, and administrators.
🚀 Tech Stack

    Backend: Django 5.0+, Django REST Framework

    Database: PostgreSQL 15+ (Neon)

    Cache: Redis (Upstash)

    Payments: Razorpay

    Email: SendGrid

    Media: Cloudinary

    Frontend: TailwindCSS, Alpine.js

    Deployment: Railway, Docker

    CI/CD: GitHub Actions

    Monitoring: Sentry

🎯 Core Features

    User Management: Registration, email verification, social login (Google/GitHub)

    Memberships: Tiered plans with recurring subscriptions

    Bookings: Personal training sessions and group classes

    Dashboards: Role-based views for Members, Trainers, and Admins

    Payments: Razorpay integration with webhook verification

    Notifications: Email and in-app alerts

    Security: Role-based access, rate limiting, brute force protection
